# GROUPDNA – WhatsApp Chat Analyzer

## Overview

GROUPDNA is a WhatsApp chat analysis project developed using Python and NumPy.

## Features

- Chat Parser
- Group Overview
- Most Active Day
- Most Active Hour
- NumPy Activity Heatmap
- Favourite Words
- Response Time Analysis
- Silent Streak Detection
- Personality Archetypes

## Technologies Used

- Python
- NumPy
- datetime

## Output

The project generates a comprehensive report showing participant activity and messaging patterns.
